# Instalação no Android

Esta versão está preparada como PWA instalável.

## Como instalar no celular Android
1. Publique a pasta no GitHub Pages.
2. Abra o link no Chrome do Android.
3. Toque no menu de três pontos.
4. Escolha "Adicionar à tela inicial" ou "Instalar app".
5. O app aparecerá com o ícone "Copa Vendas".

## Sobre APK
Este pacote não contém um APK nativo compilado, porque a compilação exige Android SDK/Gradle/apksigner.
A versão entregue aqui é a forma mais segura para uso imediato via GitHub Pages: PWA instalável com ícone, tela cheia e cache básico.
